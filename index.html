Portfolio

Full stack project

Warsen e-commerce is a full stack application with both front and back end sides. The backend itself is a RESTFUL API, since it was built as a data-on-wire model. The frontend server can and must communicate with backend server to correctly deploy the final product. All products, users, carts, and orders modules are storaged in Mongo Atlas Database service. Built with React Js, Mongo Atlas Database, Mongoose as ORM, Node Js, Express, Factory, CORS, Axios, Dotenv, Nodemailer (with Google), sensitive data encryptation, and Mocha testing.
